## Getting Started

### Prerequisites

Nodejs is required.

### Installation

Steps


1. Clone the repo
   ```sh
   git clone https://github.com/FedeHerrera10/landing-lopez-informatica
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
